# Icons needed for PWA

Please create or generate the following icons:

1. **icon-192.png** - 192x192px icon with a flower design
2. **icon-512.png** - 512x512px icon with a flower design
3. **favicon.ico** - Standard favicon

## Suggestion:

You can use an emoji-to-icon converter or create simple icons with a flower emoji (🌸) on a soft pink background (#FFE4E9).

## Quick solution:

Use a free tool like:

- https://favicon.io/favicon-converter/
- https://realfavicongenerator.net/

Upload a flower image or emoji and generate the required sizes.
